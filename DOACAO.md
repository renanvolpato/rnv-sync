# 💛 Apoie o RNV Sync

O RNV Sync é **gratuito e open source** (licença MIT). Se ele te
economiza tempo e você quiser ajudar a manter o projeto vivo —
correções, novas funcionalidades e suporte — qualquer contribuição
via **PIX** é muito bem-vinda. Não tem valor mínimo: contribua com o
que fizer sentido pra você. 🙏

## PIX (Brasil)

<img src="docs/images/pix.png" alt="QR Code PIX" width="240">

Aponte a câmera do app do seu banco para o QR acima, **ou** use o
**copia e cola** (abra o app do seu banco → PIX → "Pix Copia e Cola"
→ cole o código abaixo → informe o valor):

```
00020101021126580014br.gov.bcb.pix0136836379ba-b09f-425f-80bd-6540b6dde86d5204000053039865802BR5922RENAN A DOS S VOLPATTO6007MARINGA62070503***63045443
```

**Chave PIX (aleatória):**

```
836379ba-b09f-425f-80bd-6540b6dde86d
```

> O PIX é **somente recebimento**: o código acima serve apenas para
> te enviar uma contribuição — ninguém consegue retirar nada com ele.
> O nome do favorecido faz parte do padrão PIX e só aparece para você
> na hora de confirmar o pagamento no app do seu banco.


## Outras formas

GitHub Sponsors (apoio mensal recorrente) pode ser habilitado no
futuro — por enquanto o PIX é a forma mais direta e sem taxas.
